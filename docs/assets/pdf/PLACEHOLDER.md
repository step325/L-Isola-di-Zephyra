# PDF Downloads

Questa cartella conterrà i file PDF scaricabili dei quattro libri della Biblioteca Infinita.

## File Richiesti

Genera i seguenti PDF e inseriscili in questa cartella:

```
pdf/
├── Chronicon-Aethyris.pdf
├── De-Gentibus-Magicis.pdf
├── Il-Cataclisma-del-Nord.pdf
└── Teclis-Liberator.pdf
```

## Come Generare i PDF

Segui le istruzioni dettagliate nel file [`GENERA_PDF.md`](../../GENERA_PDF.md) nella cartella docs.

**Metodo veloce:**
1. Apri `book1.html`, `book2.html`, `book3.html`, `book4.html` nel browser
2. Per ciascuno: `Ctrl+P` → "Salva come PDF" → Abilita "Grafica di sfondo"
3. Salva qui con i nomi corretti

## Nota

I link di download sulla homepage (`index.html`) funzioneranno solo dopo aver creato e caricato questi PDF.

Se non vuoi generare i PDF subito, puoi semplicemente rimuovere i pulsanti "Scarica PDF" dall'`index.html`.
